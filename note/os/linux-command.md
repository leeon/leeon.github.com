##文件操作

------------------

### _find_

根据名字的目录或者文件

`find [dir] [-name] [name]` 

*option:* 

-  *-ls* 查找文件并显示



### _ls_
显示当前的目录下的文件

*options:* 

- *-G* 区别文件夹颜色显示
- *-a* 显示所有文件（包括隐藏）
- *-l* 显示文件的详细信息
